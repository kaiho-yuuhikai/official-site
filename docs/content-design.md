# 開邦雄飛会 公式サイト コンテンツ設計書

> 作成日: 2026-03-15
> 作成者: 神谷乗治（14期）
> 目的: 各ページに掲載すべきコンテンツの詳細設計

---

## 全体方針

- **情報の鮮度**: 固定情報（理念・会則）と流動情報（ニュース・基金状況）を分離し、更新負荷を最小化
- **データソース**: Google Docs/Sheets で運営メンバーが編集 → JSON変換 → サイト反映
- **外部連携**: note.com（記事）、Google Forms（応募）、Facebook/Instagram（SNS）を活用し、サイト自体の更新頻度を抑える
- **段階的公開**: 「準備中」で公開し、コンテンツ確定次第差し替える

```

  - 写真アップロード
  - 一言メッセージ
  - 活動可能な曜日・時間帯
  - 過去のメンター経験
```

#### 学校との日程調整フロー（サイト外運用）

```
1. メンター登録 → Google Forms → Google Sheets に蓄積
2. 学校から依頼（特設授業、探究活動等） → 渉外チームが受付
3. 渉外チームがメンターリストから適任者を選定
4. メンターに打診 → 日程調整（メール or LINE）
5. 確定後、学校に報告

※ サイト上では「メンター登録フォーム」と「学校向け依頼フォーム」の2つを設置
```

### 要対応

1. **メンター登録用 Google Forms の作成**
2. **既存メンターデータの移行**: cms-data.json に30名以上のプロフィールあり（大同窓会登壇者）→ メンター希望者を確認
3. **学校向け依頼フォームの作成**（渉外チーム窓口）
